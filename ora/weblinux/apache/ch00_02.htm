0.2. The Demonstration CD-ROM

The content of the CD-ROM that accompanies the print book is located in the directory cd_demo, and can be read by both Win32 and Unix systems. It contains the demonstration web sites referred to throughout the book. These are organized into two directories:

install/

This directory contains scripts to install the sample sites:

  • install -- Run this script to install the sites.

  • install.conf -- Unix configuration file for install.

  • installwin.conf -- Win32 configuration file for install.

sites/

This directory contains the sample sites used in the book.
