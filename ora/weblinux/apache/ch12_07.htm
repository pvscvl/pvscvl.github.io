12.7. Server-Side Scripting

There are several different modules that allow simple (or not so simple) scripts to be embedded into HTML pages. ftp://pageplus.com/pub/hsf/xssi/xssi-1.1.html is an extended version of standard SSI commands, while http://www.vex.net/php/ and http://www.neosoft.com/neoscript/ are more powerful scripting languages.
