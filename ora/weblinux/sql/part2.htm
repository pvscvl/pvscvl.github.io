Part II. Database Programming

The power of a database is realized in the tools that use it. In this section, we talk about how you build those tools using a variety of today's popular programming languages. From web-based programming to business application development, we will discuss the APIs and tools necessary for using MySQL and mSQL to their fullest potential. This second section starts with a couple of high-level chapters on database application architectures and CGI programming. The meat of the section, however, deals with programming for MySQL and mSQL in various programming languages.
