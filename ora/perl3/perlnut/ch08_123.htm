8.123. I18N::LangTags::List

Provides a lengthy and nearly comprehensive list of any language tags that you might need, from ab to zn. You should refer to the I18N::LangTags::List documentation for the list of these tags and what they represent.

I18N::LangTags::List implements the name() function that takes a language tag and returns its best attempt at an English name for it. If name() doesn't understand the language tag, it returns undef.

For example:

#!/usr/local/bin/perl -w
use I18N::LangTags::List;
 
my $tag = 'en-CA';
print "What is this tag aboot? ",
	    I18N::LangTags::List::name($tag),

"\n";
