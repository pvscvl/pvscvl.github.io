8.84. ExtUtils::MM_OS2

Provides methods for use with MakeMaker for OS2 systems. Used internally by MakeMaker if needed. (See ExtUtils::MM_Unix for documentation of the methods provided there.) Overrides the implementation of the methods, not the interface.
