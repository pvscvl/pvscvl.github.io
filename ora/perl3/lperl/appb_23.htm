B.23. Dynamic Loading

The basic idea of dynamic loading is that your program decides at runtime that it needs more functionality than what's currently available, so it loads it up and keeps running. You can always dynamically load Perl code, but it's even more interesting to dynamically load a binary extension.[419] This is how non-Perl modules are made.

[419]Dynamic loading of binary extensions is generally available if your system supports that. If it doesn't, you can compile the extensions statically -- that is, you can make a Perl binary with the extension built in, ready for use.
