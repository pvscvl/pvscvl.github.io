About Searching on Other Platforms

You'll need to get a JRE or JDK. There is an excellent list at http://java.tqn.com/msub2.htm. Once you have it, go to the netresults directory of the CD-ROM, and run 

jre -mx16m -cp . -ms8m itm.nr.serve.NRServer

If you are having any problems running the server, please:

  1. Try a demonstration program that came with your JRE or JDK to be sure it is properly installed.
  2. Run the command again as 
    jre -mx16m -cp . -ms8m itm.nr.serve.NRServer +ds +es +vs

    for maximum debugging information before contacting us.
