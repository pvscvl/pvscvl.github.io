8.148 Thread

Provides Perl multithreading support. Distributed as a beta feature with Perl 5.005 and runs only on versions of Perl that were built with thread support. Has the following functions and methods:

  • new
  • async
  • cond_broadcast
  • cond_signal
  • cond_wait
  • eval
  • join
  • lock
  • list
  • self
  • tid
