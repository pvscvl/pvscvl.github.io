Chapter 10
The CGI.pm Module
 

redirect

 print $query->redirect('url')

Generates a header for redirecting the browser.

url

The absolute URL to redirect to.

Using named parameters, the syntax is: 

print $query->redirect(-uri=>'url',
                       -nph=>1);
