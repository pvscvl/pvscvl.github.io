ObjectOutput

Name

ObjectOutput

Synopsis

Interface Name:

java.io.ObjectOutput

Super-interface:

java.io.DataOutput

Immediate Sub-interfaces:

None

Implemented By:

java.io.ObjectOutputStream

Availability:

New as of JDK 1.1

Description

The ObjectOutput interface extends the DataOutput interface for object serialization. While DataOutput defines methods for reading primitive types from a stream, ObjectOutput defines methods for writing objects and arrays of bytes.

Interface Declaration

public abstract interface java.io.ObjectOutput extends java.io.DataOutput {
  // Methods
  public abstract void close();
  public abstract void flush();
  public abstract void write(int b);
  public abstract void write(byte[] b);
  public abstract void write(byte[] b, int off, int len);
  public abstract void writeObject(Object obj);
}
