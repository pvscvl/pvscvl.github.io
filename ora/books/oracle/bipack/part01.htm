Part I: Overview

This part of the book contains a single chapter that introduces the built-in packages and provides some quick-reference tools to help you use the remainder of the book. Chapter 1, Introduction, introduces you to PL/SQL packages generally and to built-in packages specifically. It shows you how to call packaged code from your own programs and includes tips for handling the exceptions raised when you call a packaged program.
