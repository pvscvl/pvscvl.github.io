Preface
 

Versions of Oracle

The built-in packages of Oracle PL /SQL discussed in this book are installed in the Oracle database. Each version of the Oracle database comes with its corresponding version of PL /SQL and its set of built-in packages. Most of the built-in packages became available with Oracle Release 7.1. A number of new packages were provided with Oracle8. The text describes which packages are available only with specific versions of Oracle.
