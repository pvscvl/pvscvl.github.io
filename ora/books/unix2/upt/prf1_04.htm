Like a Hypertext Database

Each article doesn't define all the concepts and words used. Instead, it gives you "links" that let you get more information if you need it. It's easy to get more information when you need it but skip the link if you don't. UNIX Power Tools® uses two kinds of links: in a sentence and in the margin. For examples, see the pages before this Preface titled How to Use This Book.
