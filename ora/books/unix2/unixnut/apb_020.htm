Appendix B
Obsolete Commands
 

openwin

/usr/openwin/bin/openwin [options]

Start the OpenWindows graphical user interface environment. This environment is now considered obsolete; the preferred environment is CDE (the Common Desktop Environment), and OpenWindows will not be supported past Solaris 7. See also cde in Chapter 2, Unix Commands.
