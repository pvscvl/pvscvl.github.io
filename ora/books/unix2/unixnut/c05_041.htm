set variable = value
set variable[n] = value
set

Set variable to value, or, if multiple values are specified, set the variable to the list of words in the value list. If an index n is specified, set the nth word in the variable to value. (The variable must already contain at least that number of words.) With no arguments, display the names and values of all set variables. See also the section "Predefined Shell Variables" earlier in this chapter.

Examples

% set list=(yes no maybe)           Assign a word list
% set list[3]=maybe                 Assign an item in existing word list
% set quote="Make my day"           Assign a variable
% set x=5 y=10 history=100          Assign several variables
% set blank                         Assign a null value to blank
