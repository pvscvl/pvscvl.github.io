Chapter 12
nroff and troff
 

.af

.af r c

Assign format c to register r.

Values for c

1

0, 1, 2, etc.

001

000, 001, 002, etc.

i

Lowercase roman numerals.

I

Uppercase roman numerals.

a

Lowercase alphabetic.

A

Uppercase alphabetic.

Example

Paginate front matter using the ms macros:

.af PN i	Set page number register PN to i
