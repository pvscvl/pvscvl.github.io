.P

.P [type]

Start new paragraph. A paragraph type can be specified, overriding default. Various registers can be set to control default formats:

Pt

Paragraph type for document (default is 0).

Pi

Amount of indent (default is 3n).

Ps

Spacing between paragraphs (default is one line of white space).

Np

Set this to 1 to produce numbered paragraphs.

Type

0

Left-justified (the default).

1

Indented.

2

Indented except after displays (.DE), lists (.LE), and headings (.H).
